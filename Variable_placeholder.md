# 占位符变量解释
**所有DES描述系列所需要的统计数据，均可通过output_excel下文件获得相关统计数据，优先访问output_excel下文件，实在得不到相关统计数据，则访问original_data下的最新日期原始数据。必须保证所有统计数据可溯源，有依据，对得上**
- LINE_NUMBER:填入几号线，根据用户在欢迎页面选择的线路号，例如用户选择"7号线"则填7

- MONTH:当前系统时间是几月，则填入几月。**注意：取当前系统时间月份（如今天6月22日则填6），不要从检测数据文件名中的月份推导！示例：当前为6月则填6**

- who_you_are:分析人员姓名，由用户在Web欢迎页面（welcome.html）登录时输入。通过Flask session传递到后端，最终由Claude CLI填入报告模板report_template_all.html的{{who_you_are}}占位符（位于报告封面”分析人员”栏，第189行）。在Web服务中，用户访问根路径”/”时首先看到欢迎登录界面，输入姓名后跳转到”/upload”分析页面，后续生成报告时自动带入该姓名。

- DATE_TODAY:联网获取当前日期，格式为”2026年05月10日”

- Vehicle_NUMBER:填入车号，根据original_data文件夹下文件名，例如W102

- START_END:始末区间，根据线路询问用户始末站，例如“5号线到始末站是什么”，“赤湾-大剧院”，则填入“赤湾-大剧院”

- ALL_FILE_DATE:几个月的数据，则填入几个日期，例如“2026年1月4日、2月8日、3月10日”、“2026年3月16日、2026年4月18日”

- DATE_TEST:根据original_data文件夹下各个excel文件名，填入**最新的检测日期**（按实际日期比较，不是文件名字母排序！），格式：”2026年5月24日”

- LINE_LONG:从当前目录下的more_info.json文件中获取对应线路里程数（根据LINE_NUMBER匹配）。**禁止填"待补充"，必须从more_info.json获取真实数值！** 示例：11号线LINE_LONG=114.98

- LINE_WIDTH:从当前目录下的more_info.json文件中获取对应线路接触线线径（根据LINE_NUMBER匹配）。**禁止填"待补充"，必须从more_info.json获取真实数值！** 示例：11号线LINE_WIDTH=150

- MAX_SPEED:从当前目录下的more_info.json文件中获取对应线路最高允许速度（根据LINE_NUMBER匹配）。**禁止填"待补充"，必须从more_info.json获取真实数值！** 示例：11号线MAX_SPEED=120

- table2_1/2_2/2_3:参照output_excel下的table2_1/2_2/2_3

- DEFECT_DES:根据output_excel下的table2_2，table2_3完成描述，描述示例如下：
  本次任务发现缺陷132处，其中一级缺陷0项，二级缺陷13项，三级缺陷25项，主要缺陷类型集中在绝缘子脏污、副螺帽缺失及定位线夹卡滞，详见表2-2。

- HARDPOINT_STATIC_DES1:对全线硬点分布统计进行描述，描述示例如下：
  （1）14号线2026年1月4日、2月8日、3月10日上行硬点数量大于10g，分别为78处、14处、45处，对比2月8日硬点数量增加31处；下行硬点数量大于10g，分别为54处、7处、117处，对比2月8日硬点数量增加110处；
  - 相关硬点超限数据可以通过“output_excel/graph1_hardpoint_monthly_comparison.xlsx”“output_excel/graph3_”获得
  - **根据硬点数量变化使用不同的程度词：硬点数量变化量小于50“轻微增加”，50-100“略微增加”，硬点数目增加超过100则“增加明显”**

- HARDPOINT_STATIC_DES2:对全线硬点分布统计进行描述，描述示例如下：
  <p>（2）硬点数量集中在上行黄木岗站-罗湖北站、坑梓站-沙田站分别为15处、14处。下行坳背站-四联站、沙田站-坑梓站、宝龙站-南约站分别为30处、17处、16处，对比增加明显。</p>

  - **根据硬点数量变化使用不同的程度词：硬点数量变化量小于50“轻微增加”，50-100“略微增加”，硬点数目增加超过100则“增加明显”**

- HARDPOINT_DATA:将output_excel/HARDPOINT_DATA_UP.xlsx和output_excel/HARDPOINT_DATA_DOWN.xlsx 按照html格式插入到占位符处

- STAT_PULLOUT_IRREGULAR_COUNT:询问用户拉出值不平顺有多少处

- STAT_PULLOUT_IRREGULAR_DEC:询问用户往月拉出值不平顺有多少处，作出比较，描述该比较。描述示例：“较上次有所改善/恶化”

  <p>（2）对比2月8日导高弛度分布，上行导高弛度大于8mm分布占比为1.13%下降为0.96%；下行导高弛度大于8mm分布占比从1.98%上升为2.50%。</p>
