# W102 检测数据分析报告 — 数据处理规范

## 一、通用数据预处理

| 规则 | 说明 |
|---|---|
| 站区为空的行 | **全部脚本统一过滤**：`dropna(subset=['站区'])` |
| 数据目录结构 | `original_data/上行/` 和 `original_data/下行/` 子目录，方向由目录名确定 |
| 文件名格式 | `全线路原始数据_{线名}_{车号}_{行别}_{日期}.xlsx` |
| 线路识别 | 从文件名提取，如 `深圳地铁5号线`、`深圳地铁11号线` |
| 5号线/11号线特殊规则 | 导高驰度和磨耗宽度计算时，仅统计 **导高 < 4400mm** 的数据点 |

---

## 二、硬点（Hardpoint）

### 阈值
- **7g = 70 m/s²**（g 取 10 m/s²）

### 超限判定（OR 逻辑）
- 一行数据中，**硬点1(m/s²) > 70 或 硬点2(m/s²) > 70**，该行即算一个超限数据点
- 每行最多算一次

### 分级标准（generate_table2_1.py）

| 线路 | Ⅰ级 | Ⅱ级 | Ⅲ级 |
|---|---|---|---|
| **标准线路** | 70 < val ≤ 130 | 130 < val ≤ 200 | val > 200 |
| **11号线** | 100 < val ≤ 150 | 150 < val ≤ 250 | val > 250 |

- 分级时取该行所有硬点列的**最大值**
- 区间均为**前开后闭** `(a, b]`

### Top-10 导出（export_top10_tables.py）
- 按 **硬点2(g)** 降序排列
- 杆号互不相同（去重取最大）
- 单位转换：m/s² → g **除以 9.8**

---

## 三、导高驰度（Height Diff）

### 计算方式
- 按 **"站区" + "杆号"**（字符串拼接）分组
- 组内计算 `max(导高(mm)) - min(导高(mm))`
- 5号线/11号线：仅统计导高 < 4400mm 的数据点

### 差值分布区间（graph5 饼状图）
- 区间：`0-4, 4-6, 6-8, 8-10, 10-12, 12-15, 15-100`（单位 mm）
- 区间语义：**前开后闭** `(a, b]`
- 仅使用**最新月份**数据

### 分级标准（generate_table2_1.py）

| 分级 | 条件 |
|---|---|
| Ⅰ级 | 10 < diff ≤ 15 |
| Ⅱ级 | 15 < diff ≤ 25 |
| Ⅲ级 | diff > 25 |

- 区间均为**前开后闭** `(a, b]`

### Top-10 导出（export_top10_tables.py）
- 按高差降序，取前 10
- 杆号互不相同（去重取最大）
- 仅保留 **高差 ≤ 30mm**
- 11号线额外：排除特定杆号范围 刚柔过渡区域
  ranges = ["LC25-29~19", "LC13-02~10", "LC11-01~13", "LC01-01~10"]
  singles = ["LC004", "LC110", "LC001"]
  
  ranges = ["RC02-01~10", "RC12-05~13", "RC14-01~10", "RC26-24~30"]
  singles = ["RC004"]

## 四、接触压力（Contact Pressure）

### 阈值

| 参数 | 值 | 说明 |
|---|---|---|
| 上限阈值 | **130 N** | 压力 > 130N 视为超限 |
| 下限阈值 | **100 N** | 压力 < 100N 视为偏低 |
| 最低有效值 | **10 N** | 压力 ≤ 10N 视为接触异常/弓网离线，**不予统计** |

### 超限判定（graph6 / generate_table2_1.py）
- **高超限**：压力 > 130 N
- **低超限**：10 N < 压力 < 100 N
- 压力 ≤ 10 N：**全部统计排除**

### 分级标准（generate_table2_1.py）

#### 接触压力 < 100N（低压力）

| 分级 | 条件 |
|---|---|
| Ⅰ级 | 80 ≤ val < 100 |
| Ⅱ级 | 50 ≤ val < 80 |
| Ⅲ级 | 10 < val < 50 |

- 区间均为**前闭后开** `[a, b)`

#### 接触压力 > 130N（高压力）

| 分级 | 条件 |
|---|---|
| Ⅰ级 | 130 < val ≤ 180 |
| Ⅱ级 | 180 < val ≤ 220 |
| Ⅲ级 | val > 220 |

- 区间均为**前开后闭** `(a, b]`

### Top-10 导出（export_top10_tables.py）
- 按压力和降序，取前 10
- 杆号互不相同（去重取最大）
- 仅保留 **压力 ≥ 130N**

## 五、磨耗宽度（Wear Width）

### 阈值
- 关注范围：**≥ 8 mm**

### 分布区间（graph7 折线图）
- 区间：`8-10, 10-12, 12-14, >14`（单位 mm）
- 区间语义：**前开后闭** `(a, b]`（`>14` 为 `(14, +∞)`）

### 分级标准（generate_table2_1.py）

#### 标准线路

| 分级 | 条件 |
|---|---|
| Ⅰ级 | 8 < val ≤ 10 |
| Ⅱ级 | 10 < val ≤ 13 |
| Ⅲ级 | val > 13 |

#### 1号线 / 5号线

| 分级 | 条件 |
|---|---|
| Ⅰ级 | 7 < val ≤ 8 |
| Ⅱ级 | 8 < val ≤ 11 |
| Ⅲ级 | val > 11 |

- 区间均为**前开后闭** `(a, b]`

### Top-10 导出（export_top10_tables.py）
- 按磨耗宽度降序，取前 10
- 杆号互不相同（去重取最大）
- 5号线/11号线：排除导高 ≥ 4400mm 的数据点

## 八、表格输出规范

### table2_1（几何参数缺陷统计）
- 来源：`generate_table2_1.py`
- 数据：仅取**最新日期**的上下行文件
- 表头列顺序：类别、问题类型、单位、**Ⅲ级、Ⅱ级、Ⅰ级**

### table2_2 / table2_3（缺陷总览）

- 一级最严重

### Top-10 表格（export_top10_tables.py）
- 处理：**最新日期**的上下行文件
- 输出 4 对文件（上下行各一份）

## 九、报告生成规范

| 规则 | 说明 |
|---|---|
| MONTH | **当前系统月份**，非文件名中的月份 |
| DATE_TEST | 按日期比较取**最新**（非文件名字母排序） |
| 行别用语 | "上行" / "下行"，**禁用** "上行业" / "下行业" |
| 单位 | 硬点=g，压力=N，磨耗=mm，导高=mm |
| LINE_LONG/LINE_WIDTH/MAX_SPEED | 从 **more_info.json** 获取，**禁止**填"待补充" |
| 变化幅度描述 | <50="轻微增加"，50-100="略微增加"，>100="增加明显" |
| HTML 表格序号列 | **不含**"序号"列 |
| 缺陷严重程度 | 一级 > 二级 > 三级（一级最严重） |

## 十、区间开闭方向速查

| 数据类型 | 区间语义 | 示例 |
|---|---|---|
| **导高差值** (graph5 / 分级) | 前开后闭 `(a, b]` | `(4, 6]` — 不含 4，含 6 |
| **磨耗宽度** (graph7 / 分级) | 前开后闭 `(a, b]` | `(8, 10]` — 不含 8，含 10 |
| **拉出值** (graph4 / graph2 / graph9) | 前闭后开 `[a, b)` | `[-250, -240)` — 含 -250，不含 -240 |
| **硬点分级** | 前开后闭 `(a, b]` | `(70, 130]` — 不含 70，含 130 |
| **压力低分级** | 前闭后开 `[a, b)` | `[50, 80)` — 含 50，不含 80 |
| **压力高分级** | 前开后闭 `(a, b]` | `(130, 180]` — 不含 130，含 180 |
